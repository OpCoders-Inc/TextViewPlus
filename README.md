# TextViewPlus

TextView+ is an Application for C64 OS for viewing text files in PETSCII, ASCII or MText format. It has more memory to view larger files than can be viewed with TextView in File Manager.

## Installation
- copy the `TextView+.car` to the Application directory using File Manager
- double-click the `TextView+.car` in File Manager
- click the EXTRACT button when the INSTALLER utility opens

After the extract is complete, you should have a new Application called `TextView+` and then you can double-click the application to open it.
